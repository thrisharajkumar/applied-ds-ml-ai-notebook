# Applied DS, ML & AI Notebook

An ongoing, public notebook tracking my learning and work across data science, machine learning, and AI — daily logs, paper notes, and project reasoning, organized around a four-quadrant map of the field.

**Live site:** https://thrisharajkumar.github.io/applied-ds-ml-ai-notebook/

## About

I'm Thrisha Rajkumar, a Data Scientist / ML Engineer working across clinical AI, time-series modelling, and applied deep learning — currently a Machine Learning Research Intern at the University of Bristol, with prior work on an NIHR-funded clinical AI platform piloted with NHS stroke patients, and two 2026 first-author conference papers (HEALTHINF, Barcelona · IPEC-ECCE Asia, Nagasaki).

This repo is a continuous, dated record of what I'm reading, building, and learning — not a polished, finished portfolio piece. Expect it to look "in progress" most of the time, because it is.

## Running it locally

This site is built with [MkDocs Material](https://squidfunk.github.io/mkdocs-material/). To preview it yourself:

```bash
pip install mkdocs-material
mkdocs serve
```
Then open http://127.0.0.1:8000
